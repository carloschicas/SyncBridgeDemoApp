## [0.1.0] - 2026-04-16

### Added

- Proyecto base inicializado con Android Studio (Kotlin + Jetpack Compose)
- Configuración de `minSdk 26`, `targetSdk 36`, `compileSdk 36`
- Dependencias de Room (`room-runtime`, `room-ktx`, `room-compiler` via KSP)
- Dependencias de Hilt (`hilt-android`, `hilt-android-compiler`, `hilt-navigation-compose`)
- Dependencias de Coroutines (`kotlinx-coroutines-android`)
- Plugin KSP para procesamiento de anotaciones en tiempo de compilación
